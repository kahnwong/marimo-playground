# Marimo Playground

Ref: <https://github.com/marimo-team/marimo>

## Quickstart

```bash
uv run marimo tutorial intro
```

## Usage

```bash
# start a file browser
uv run marimo edit

# start marimo against a specific file
uv run marimo edit main.py

# run as python file
uv run python3 main.py
```
